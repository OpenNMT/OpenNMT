onmt.BiEncoder

BiEncoder is a bidirectional Sequencer used for the source language.

netFwd

h_1 => h_2 => h_3 => ... => h_n
 |      |      |             |
 .      .      .             .
 |      |      |             |
h_1 => h_2 => h_3 => ... => h_n
 |      |      |             |
 |      |      |             |
x_1    x_2    x_3           x_n

netBwd

h_1 <= h_2 <= h_3 <= ... <= h_n
 |      |      |             |
 .      .      .             .
 |      |      |             |
h_1 <= h_2 <= h_3 <= ... <= h_n
 |      |      |             |
 |      |      |             |
x_1    x_2    x_3           x_n

Inherits from onmt.Sequencer.
